质因子分解

发表于 2017-12-19 | 分类于 编程 | 阅读次数

将一个数分解质因子(因为一个数字要么是质数要么是能分解为质数的乘积)，或者求一个数字的质因子算法—- Pollard Rho 因数分解。

  • 如果这个质数恰等于n，则说明分解质因数的过程已经结束，打印出即可。
  • 如果n<>k，但n能被k整除，则应打印出k的值，并用n除以k的商,作为新的正整数你n,
    　重复执行第一步。

  • 如果n不能被k整除，则用k+1作为k的值,重复执行第一步。

    #include<cmath>
#include<stdio.h>
using namespace std;
void primeDecomposition(int n){
    int i=0;
    if(n==2){
        printf("2");
 }
    else if(n==1){
     printf("1不能再分解!");
 }
 else{
      for(i=2;i<=sqrt(n);i++){
         while(n!=i){
              if(n%i==0){
                   printf("%d\t",i);
                    n=n/i;
            }
              else{
                 break;
             }
            }
     }
     printf("%d\n",n);
 }
}
int main(){

    int n;
    printf("请输入要分解的数字n:\n");
    scanf("%d",&n);
    primeDecomposition(n);
    }
    return 0;
}

对n进行质因数分解，从2开始依次增加找到一个该数最小的质因子k，直到增加到该数的算术平方根（因为如果一个数直到他的算数平方根为止都没有质因子，后边也不会有了）。
找到其最小的质因子之后，将其打印出来，然后用n/k将结果赋值给n，紧接着上一步的k往后继续进行上述步骤
如果直到算数平方根都没有质因子，便直接打印出该数，此时该数是质数。
